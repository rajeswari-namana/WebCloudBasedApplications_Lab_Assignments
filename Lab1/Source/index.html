Creation Of Git Repository

Git & GitHub:

Git is a version control system for tracking changes in computer files and coordinating work on those files among multiple people. It is primarily used for source code management in software development, but it can be used to keep track of changes in any set of files. As a distributed revision control system it is aimed at speed, data integrity, and support for distributed, non-linear workflows. GitHub is a Git repository hosting service, but it adds many of its own features. While Git is a command line tool, GitHub provides a Web-based graphical interface. It also provides access control and several collaboration features, such as a wikis and basic task management tools for every project.

Repository:

It is a directory or storage space where your project files are kept. It can be local to a folder on your computer or it can be storage space on GitHub or any other host. There are two types of repositories:

  • Central/Remote repository(hosted by a website)
  • Local Repository(on your computer)

Steps for creating Git Repository:

  1. GitHub Sign up:

    2. First you need to create a GitHub account at the website "https://github.com/" where you need to provide a username and password along with your email address.

  2. GitHub Sign in:

    3. After verification of your account through your email, sign in to GitHub account

  3. Remote Repository Creation:

    4. After signing in:

    1. you will see a green button on the right saying "New repository" or else if you click on the "+" at the top right, there will also be New Repository option, click on any one of them.

    2. And in the next page fill in details such as repository name, choosing as public or private etc and click on create repository

    3. And remote repository will be created as show in the below screenshot.

  4. Local Repository Creation:

In order to clone the git repo to your desktop you need to install either GitHub Desktop app or Git Shell, both can be downloaded from "https://desktop.github.com/". Lets consider using GitHub Desktop app, after installing it you have to sign in with the same credentials as your GitHub account and then you will be able to clone the remote repo to your local machine by opening the app. Open the GitHub desktop app:

  1. Then press on "+" option on the top left corner, where you can see your created repo name and click clone.

  2. Choose the path where your repo will be cloned and press ok button.

  3. If you go to that path you can observe that your local repo has been created or cloned there.
